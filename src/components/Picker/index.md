# Picker 选择器

提供多个选项集合供用户选择，支持单列选择和多列级联，通常与[弹出层](../feedback/popup)组件配合使用。

### API

| 参数      | 说明            | 类型          | 默认值 |
| --------- | --------------- | ------------- | ------ |
| className | 自定义 class 名 | string        | -      |
| style     | 自定义样式      | CSSProperties | -      |
| zIndex    | z-index 值      | number        | -      |
