包裹住 Button 组件，根据子元素的多少，进行 flex 布局

| 参数      | 说明                                         | 类型   | 默认值      |
| --------- | -------------------------------------------- | ------ | ----------- |
| className | 自定义 class 名                              | string | -           |
| variant   | 设置按钮形态，可选值 contained、 outlined    | string | 'contained' |
| color     | 设置颜色，可选值为 'blue'、'orange'、'green' | string | 'blue'      |
