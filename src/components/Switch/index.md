## API

| 属性           | 说明                   | 类型                        | 默认值 |
| -------------- | ---------------------- | --------------------------- | ------ |
| checked        | 值                     | boolean                     | -      |
| defaultChecked | 初始值                 | boolean                     | -      |
| disabled       | 是否禁用               | boolean                     | false  |
| onChange       | 值变化时触发的回调函数 | `(value?: boolean) => void` | -      |
