### API

| 属性            | 说明                                   | 类型      | 默认值 |
| --------------- | -------------------------------------- | --------- | ------ |
| dashed          | 是否使用虚线                           | boolean   | false  |
| hairline        | 是否使用 0.5px 线                      | boolean   | true   |
| contentPosition | 内容所在位置，可选 left，center，right | string    | center |
| children        | 组件中的内容                           | ReactNode | -      |
