**Props**

| 属性            | 说明                          | 类型                | 默认值 |
| --------------- | ----------------------------- | ------------------- | ------ |
| className       | 自定义 Class 名               | string              | -      |
| size            | 大小，可选值 sm、md、lg、auto | string              | 'md'   |
| color           | 颜色                          | string              | 'blue' |
| icon            | svg 文件                      | any                 | -      |
| style           | 自定义样式                    | React.CSSProperties | -      |
| keepOriginColor | 是否保持原有颜色              | boolean             | false  |
| onClick         | 点击图标后的回调              | -                   | -      |
